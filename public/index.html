NHTSA Vehicle Data MCP Server

This server provides MCP access to the NHTSA vehicle database APIs.

Connection Types

  • Standard MCP: {{HOST_URL}}/mcp
  • Server-Sent Events (SSE): {{HOST_URL}}/sse

Project Structure

The server is organized using a modular approach:

  • API Layer: Handles communication with the NHTSA API
  • MCP Tools: Categorized by functionality (VIN, Makes, Models, etc.)
  • MCP Resources: URI-addressable vehicle data resources

Available Resources

  • nhtsa://api-info - API documentation
  • nhtsa://makes - List of all vehicle makes
  • nhtsa://makes/{make} - Information about a specific make
  • nhtsa://manufacturers - List of all manufacturers
  • nhtsa://manufacturers/{manufacturer} - Information about a specific manufacturer
  • nhtsa://variables - List of all vehicle variables

Tools Categories

  • VIN Decoding: Decode vehicle identification numbers
  • WMI: World Manufacturer Identifier tools
  • Makes & Models: Vehicle make and model lookup
  • Manufacturers: Manufacturer information
  • Parts: Vehicle parts data
  • Vehicle Variables: Standard vehicle specification variables
  • Canadian Specifications: Canadian vehicle data

For more information, visit the GitHub repository.
